# Boilerplate Website

A modern, responsive website boilerplate built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Fully responsive layout that works on all devices
- **Navigation Menu** - Sticky navigation with mobile hamburger menu
- **Hero Section** - Eye-catching landing section with call-to-action
- **About Section** - Information about your business
- **Services Section** - Display your services in a grid layout
- **Contact Form** - Functional contact form for user inquiries
- **Modern Styling** - Clean and professional CSS with smooth transitions
- **Cross-browser Compatible** - Works across all modern browsers

## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # CSS styles and responsive design
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content, colors, and branding to match your needs
3. Replace placeholder text with your own content
4. Modify the color scheme in `styles.css` by changing the CSS variables in the `:root` section

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    --dark-color: #212529;
    --light-color: #f8f9fa;
}
```

### Sections
Each section is easily customizable:
- **Hero**: Modify the heading and call-to-action button
- **Services**: Add or remove service cards
- **Contact Form**: Customize form fields as needed

## Browser Support

## Tips

- The website uses a mobile-first responsive design approach
- The navigation menu automatically converts to a hamburger menu on mobile devices
- All links have smooth scroll behavior
